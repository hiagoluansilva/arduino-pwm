# arduino-pwm

LED brightness control with smooth fade on ESP32 using PWM (ledcWrite).

**Channel:** 0 · **Frequency:** 10 kHz · **Resolution:** 10-bit · **Pin:** GPIO 2

## Usage

1. Open `pwm.ino` in Arduino IDE
2. Select **ESP32 Dev Module**
3. Compile and flash
